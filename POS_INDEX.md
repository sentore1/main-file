## 📊 Features Overview

All documents cover these 4 main features:

### ✏️ Feature 1: Editable Item Prices
- Adjust prices at checkout
- Different prices for different customers
- Original product prices unchanged

### 📝 Feature 2: Item Notes/Comments
- Add custom notes to items
- Specify variants (e.g., "Indian Rice")
- Notes printed on receipts

### 💰 Feature 3: Partial Payments
- Accept partial payments
- Track balance due
- Order status: 'partial'

### ➕ Feature 4: Add Customer from POS
- Create customers without leaving POS
- Quick customer registration
- Immediate availability
